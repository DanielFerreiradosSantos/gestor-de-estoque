# Sistema de Gestão de Estoque

## Descrição
Sistema desenvolvido em Python para controle de estoque de produtos.

## Objetivo
Permitir cadastro, consulta e movimentação de produtos em estoque.

## Funcionalidades
- Cadastro de produtos
- Entrada de estoque
- Saída de estoque
- Consulta de produtos
- Alerta de estoque baixo
- Listagem de produtos

## Tecnologias utilizadas
- Python

## Como executar
1. Instale o Python
2. Execute o arquivo:

```bash
python gestao_estoque_seu_nome.py
